# **App Name**: Facility AI

## Core Features:

- Authentication: User authentication with login, SSO, and password reset functionalities.
- Dashboard: Dashboard displaying KPIs, alerts, and dynamic lists of critical equipment and urgent tickets.
- Sites & Buildings: Management of sites and buildings, including details, hierarchy, and equipment status.
- Equipment Management: Management of equipment (assets) with metadata, location, documents, sensor data display, and historical records.
- Ticket Management: Ticket creation and management using a Kanban board, including assignment, SLA tracking, and media upload.
- Audits & Compliance: Mobile-friendly audit forms for compliance checks with photos, annotations, and action recommendations.
- AI-Powered Report Generation: AI tool for generating reports that will summarize key findings and action items from the audit data.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and reliability.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Vibrant orange (#FF9800) for CTAs and highlights, ensuring important elements stand out.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable appearance.
- Use a consistent set of professional icons throughout the application to aid in navigation and comprehension.
- Mobile-first design with a responsive layout to ensure seamless user experience across devices.
- Subtle transitions and animations to provide feedback and enhance user engagement without being distracting.
